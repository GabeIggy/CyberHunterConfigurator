FAQ

How can apply the configuration to my game?

Make sure the game is closed. Go to the Game Directory, search the config.ini file and then replace the section you want. ([graphic_setting] or [sensitive_setting]) If the section is missing from the file just append it at the end of the file. Save and start the game.

What motivated you to make this?

Mostly love for the game and I wanted to help friends to find their perfect sensitivity setting, making it public means it will help more people.

How is the sensitivity calculated?

The game uses a "percentage" meter that's is not standard. The best way you can find your perfect sensitivity is converting it from another game. This website may help you to find the correct value.

Is it safe to use?

As long you do not use the unsupported options you should be safe. Modifying the .ini file to access the "hidden" options may be considered cheating according to NetEase but nobody in the community has been banned so far. Still, I won't take responsibility if anything happens.

Is this considered cheating?

This does not alter the game process nor it injects into memory to give you an unfair advantage, so in most games this wouldn't be considered cheating, but you know, NetEase. Refer to the previous point.
